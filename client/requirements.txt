# Cell Counts client — install with:
#   pip install --user -r requirements.txt
#
# (tkinter ships with the standard python.org Windows/macOS installer, no separate
# install needed.)
#
# numpy, tifffile, and pillow are a real departure from Phase 1's "just requests"
# story: the server no longer generates a preview image for the client, so the
# client now loads and contrast-stretches the original TIFFs itself to render the
# review screen. All three have prebuilt wheels on Windows/macOS — no compiler, no
# admin rights needed, same install story as before, just one `pip install` away.
#
# scikit-image (coexpression.py's polygon-overlap rasterization) and openpyxl
# (XLSX export) round out the list for the same reason: both ship prebuilt wheels,
# no compiler needed, and both were chosen specifically to avoid heavier
# alternatives (shapely, pandas) for what's a small slice of what each offers.
#
# websockets (ws_client.py) is the sync client for /ws/jobs -- job status is
# pushed to the client instead of polled, so no separate polling library or
# asyncio event loop is needed here; websockets.sync.client fits this
# codebase's plain-threading style directly.
requests
numpy
tifffile
pillow
scikit-image
openpyxl
websockets
